# Pi Payroll

Pi Payroll enables employers to pay workers using Pi.

Supported:

- Full-time jobs
- Freelance work
- Service providers
- Contractors
- Gig economy

Features:

- Pi Payments
- Escrow
- Payroll history
- Automated payouts

Benefits:

- Fast payments
- Borderless payments
- Lower transaction costs

Future:

Native payroll infrastructure for the Pi Ecosystem.
