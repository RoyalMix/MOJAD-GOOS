# MOJAD AI Router Rules

Every user request passes through the AI Router.

AI Router never contains business logic.

Business logic belongs to Engines.

AI Router only decides:

• Which Engine to use

• Which Engines cooperate

• Execution order

• Result aggregation

• Personalization

• Context preservation

Every response must be explainable.

Every routing decision is auditable.

Every routing decision is logged.

Every routing decision improves future routing.

AI Router supports Pi Identity.

AI Router supports multilingual interaction.

AI Router supports offline recovery.
