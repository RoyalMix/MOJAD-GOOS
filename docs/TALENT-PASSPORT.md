# Talent Passport

Talent Passport is the universal identity layer of MOJAD GOOS.

Every user has:

- Profile
- Skills
- Experience
- Certifications
- Languages
- Trust Score
- Reputation Score
- Opportunity Score
- Portfolio
- Pi Verification

Purpose:

Create one portable opportunity identity usable across all MOJAD modules.

Modules using Talent Passport:

- JobNet
- ServiceNet
- InvestNet
- CommunicationNet
- Creator Economy
- Pi Economy

Future:

Global Talent Passport for the Pi Ecosystem.
